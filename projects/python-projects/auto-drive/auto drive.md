<b>auto drive is a tool used for uploading file to a google drive with google api for automating system backup.</b>

### design

- automatic backup.
- will parse the local backup folder and backup only new files and exclude already backuped files.
- CLI based tool.

### modules used

1. google api.
2. os.

### working

- uses google api to authenticate google account.
- upload file from local directory to google drive with google api.
- uses os module to parse files.
