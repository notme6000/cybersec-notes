# Motives, Goals & Objectives of security Attacks

_Motives = Why_
_Goals = What_ 
_Objectives = How_

### 1.Motives

- _Why attackers launch attacks_ 

1. Financial gains
    - Stealing money (bank fraud, crypto theft)
    - Ransomware payments
    - Selling stolen data on the dark web

    ![[../../../images/financial-gains-hacking.png | 500]]

2. Political or ideological (Hacktivism)
    - Protest or activism
    - Defacing websites
    - Leaking sensitive information

    ![[../../../images/hacktivists.png | 500]]

3. Espionage (corporate/Nation-state)
    - Stealing trade secrets
    - Surveillance and intelligence gathering
    - Gaining competitive advantage
4. Revenge or personal grievance
    - Disgruntled employees
    - Personal vendettas
    - Insider threats
5. Flame, challenege, or curiosity
    - Proving technical skills
    - Gaining recognition in hacker communities
    - "Just of fun" Attacks

### 2.Goals

- _What attackers want to achieve_

1. Unauthorized access
    - Breaking into systems, networks, or accounts.
2. Data theft
    - Stealing personal, financial, or classified data
3. Data manipulation
    - Alerting records
    - Injecting false information
4. Service disruption
    - Making systems unavailable (DoS/DDoS)
    - Damaging business operations
5. Control of systems
    - Installing backdoors
    - Creating botnets
    - Remote command execution

### 3.Objectives of Security Attacks

- How goals are technically carried out

1. Compromise confidentiality
    - Eavesdropping
    - Data exfiltration
    - Password sniffing
2. Compromise integrity
    - data tampering
    - malware injection
    - man-in-the-middle attacks
3. Compromise availability
    - flooding resources
    - crashing servers
    - denial-of-service attacks

### Real World Attack example mapped

- #### Financial gains -> Ransomware attacks
    - _Motive_
        - Make money through extrotion
    - _Goals_
        - Force victims to pay ransom 
    - _Objectives_
        - Encrypt critical files
        - Block access to systems
        - Threaten permanent data loss
