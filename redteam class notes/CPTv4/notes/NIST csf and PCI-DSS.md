## NIST Cybersecurity Framework (NIST CSF) 

- the _NIST Cybersecurity Framework (CSF)_ is a voluntary framework developed by the _National Institute of Standards and Technology_ to help organizations manage and reduce cybersecurity risk.

- it is widely used across 
    - Government agencies
    - Critical infrastructure
    - Enterprises
    - small & medium business
    - Global organizations (even outside the US)

- latest major version: NIST CSF 2.0 (2024 update)

[NIST CSF](https://nvlpubs.nist.gov/nistpubs/CSWP/NIST.CSWP.29.pdf) 

## What is NIST CSF

- _NIST Cybersecurity Framework_ is a risk-based cybersecurity framework that provides:
    - A common security language
    - A structured approach to cyber risk management
    - A maturity improvement path
    - Alignment with other standards (ISO 27001, SOC 2, COBIT, etc)

## PCI DSS (payment card industry Data Security Standard)

- _PCI DSS (payment card industry Data Security Standard)_ is a global security standard designed to protect cardholder data and reduce payment card fraud.

- It is developed and managed by the _PCI Security Standards Council_, which was founded by major card brands:
    - visa inc
    - mastercard
    - american express
    - discover financial services
    - JCB Co., Ltd

- Latest version: PCI DSS v4.0

[PCI DSS](https://www.pcisecuritystandards.org/standards/pci-dss/)

### Key objectives

- _protects_

- __cardholder data (CHD)__
    - primary account number (PAN)
    - cardholder name
    - expiration date 
    - service code

- __sensitive authentication data (SAD)__
    - CVV/CVC
    - full magnetic stripe data
    - PIN
