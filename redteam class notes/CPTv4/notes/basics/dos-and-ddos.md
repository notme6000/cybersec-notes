# DoS and DDoS Attacks

## 1. Denial of Service (DoS)

![Image](https://www.cloudflare.com/img/learning/ddos/http-flood-ddos-attack/http-flood-attack.png)

### Definition

A **Denial of Service (DoS)** attack is a cyberattack where a single computer sends a large amount of fake traffic to a server or network to make it unavailable to users.

### How It Works

* One attacker targets one server.
* The attacker floods the server with excessive requests.
* The server becomes slow or crashes.
* Legitimate users cannot access the service.

### Example

In 2000, a teenager launched DoS attacks that temporarily shut down major websites like Yahoo.

### Key Points

* Single source attack
* Easier to detect
* Limited scale compared to DDoS



## 2. Distributed Denial of Service (DDoS)



![Image](https://www.researchgate.net/publication/261453345/figure/fig1/AS%3A624281406955522%401525851694736/Structure-of-Distributed-Denial-of-Service-Attack.png)


### Definition

A **Distributed Denial of Service (DDoS)** attack is a cyberattack where multiple infected computers (often forming a botnet) attack a single target at the same time.

### How It Works

* Many infected devices send traffic simultaneously.
* The server becomes overloaded.
* Website or online service becomes unavailable.

### Example

The Mirai botnet carried out massive DDoS attacks in 2016, affecting websites and internet services worldwide.

### Key Points

* Multiple sources
* Harder to stop
* Larger and more powerful attack
