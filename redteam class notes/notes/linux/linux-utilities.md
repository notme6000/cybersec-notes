**Linux utilities** are small programs or tools that perform specific system tasks in a Linux operating system. They are usually command-line based and help users manage files, processes, text, disks, networks, and system information.

Think of them as building blocks — each utility does one job well.



## 1. File & Directory Utilities

These help you work with files and folders.

Examples:

* `ls` – lists files and directories
* `cp` – copies files
* `mv` – moves or renames files
* `rm` – deletes files
* `mkdir` – creates directories
* `chmod` – changes file permissions



## 2. Text Processing Utilities

Used to view, search, and manipulate text.

Examples:

* `cat` – displays file contents
* `grep` – searches text
* `awk` – processes structured text
* `sed` – edits text streams
* `sort` – sorts lines
* `wc` – counts words/lines



## 3. Process Management Utilities

Used to control running programs.

Examples:

* `ps` – shows running processes
* `top` – real-time process viewer
* `kill` – stops a process
* `nice` – changes process priority



## 4. Disk & System Utilities

Help manage storage and system info.

Examples:

* `df` – shows disk usage
* `du` – shows directory size
* `mount` – mounts filesystems
* `free` – shows memory usage
* `uname` – system information



## 5. Networking Utilities

Used for network-related tasks.

Examples:

* `ping` – checks connectivity
* `curl` – transfers data from URLs
* `ssh` – remote login
* `netstat` – network statistics



### In Simple Terms

Linux utilities are command-line tools that help you control and manage the system. They are powerful because they can be combined together using pipes (`|`) to perform complex tasks.
