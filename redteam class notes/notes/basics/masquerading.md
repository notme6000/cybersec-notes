## Masquerading

### Definition

Masquerading is a technique where an attacker impersonates a legitimate user, device, or system to gain unauthorized access.

### How It Works

Attackers obtain valid credentials through:

* Phishing
* Credential stuffing
* Data breaches
* Social engineering

They then use these credentials to appear as an authorized entity within a system.

### Objectives

* Access confidential data
* Perform fraudulent transactions
* Escalate privileges
* Disrupt operations

### Example

An attacker obtains an employee’s login credentials via phishing.
They log into the company’s internal network as that employee and download sensitive customer data.

### Types of Masquerading

* Email spoofing (pretending to be a trusted sender)
* IP spoofing
* Session hijacking
* Fake software updates

### Prevention

* Multi-factor authentication
* Strong password policies
* Account monitoring and anomaly detection
* Access control and least privilege principles
