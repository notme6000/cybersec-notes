## 1) Typosquatting

### Definition

Typosquatting is a cyberattack technique where attackers register domain names that are slight misspellings or variations of legitimate websites to deceive users into visiting malicious sites.

### How It Works

Attackers exploit common typing errors such as:

* Misspelled words (gooogle.com instead of google.com)
* Missing letters (facbook.com)
* Extra letters (amazoon.com)
* Swapped letters (goggle.com)
* Different domain extensions (.net instead of .com)

When users mistakenly enter the incorrect URL, they are redirected to a malicious site controlled by the attacker.

### Objectives

* Phishing for login credentials
* Installing malware
* Displaying malicious advertisements
* Stealing financial information

### Example

A user intends to visit a bank website:
`www.securebank.com`

An attacker registers:
`www.securbank.com`

The fake site looks identical to the legitimate one and prompts the user to enter login details, which are then captured by the attacker.

### Prevention

* Use bookmarks instead of typing URLs manually
* Enable browser phishing protection
* Monitor and register similar domain names (for organizations)
* Use multi-factor authentication
