# Social Engineering Attacks

## Definition

Social engineering attacks are cyberattacks that manipulate people into revealing confidential information or performing actions that compromise security. Instead of attacking computers directly, attackers exploit human psychology such as trust, fear, urgency, or curiosity.



## 1. Phishing

![Image](https://safecomputing.umich.edu/sites/default/files/Weblogin-Safe%26Warning.png)

![Image](https://learn-attachment.microsoft.com/api/attachments/cfd461ad-5a5d-4baf-a271-8f6422bd693f?platform=QnA)

### Description

Phishing is when attackers send fake emails or messages pretending to be from trusted organizations.

### Goal

* Steal passwords
* Collect credit card details
* Gain account access

### Example

An email pretending to be from PayPal asking the user to verify their account details.



## 2. Spear Phishing

### Description

A targeted form of phishing aimed at a specific person or organization.

### Goal

* Steal sensitive company data
* Trick employees into transferring money

### Example

An email sent to a company accountant pretending to be the CEO requesting urgent payment.

### Goal

* Obtain bank details
* Collect OTP codes

### Example

A caller pretending to be from State Bank of India asking for ATM details.



## 4. Smishing (SMS Phishing)

### Description

Fraudulent text messages that trick victims into clicking malicious links.

### Goal

* Install malware
* Steal login credentials

### Example

A text message claiming you won a prize and asking you to click a suspicious link.



## 5. Pretexting

### Description

The attacker creates a fake identity or story to gain trust.

### Goal

* Access confidential information
* Gain physical access to restricted areas

### Example

Someone pretending to be an IT technician asking for your system password.



## 6. Baiting

![Image](https://answers.webroot.com/Webroot/Images/browserpopups_1.jpg)

### Description

The attacker offers something attractive to trick victims.

### Goal

* Install malware
* Gain unauthorized access

### Example

An infected USB drive left in a public place labeled “Salary Details”.



## 7. Tailgating (Piggybacking)

### Description

An unauthorized person follows an authorized person into a restricted area.

### Goal

* Physical access to secure systems

### Example

An attacker entering an office building by following an employee through a secure door.



# Common Psychological Techniques Used

* Creating urgency
* Creating fear
* Offering rewards
* Pretending authority
* Building trust
