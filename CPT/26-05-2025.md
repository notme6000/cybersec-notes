
## proxies

- that comes between the client and the destination
### different types of proxies
- VPN is forward proxy
#### elite proxy
- high anonimity proxy
- hard to find that the traffic is comming from a server
#### anonimous proxy
- can find the traffic is form a proxy server
- hides the ip
#### socks proxy
- eg : proxychains
- works in lowerlevel session level
- high anonimity
#### caching proxy
- eg : squid proxy
- cache proxy saves the frequently visited webserver and site for faster browsing.
- fast
#### transparent proxies
- it is used to block certain sites
- it cannot be found that the system is using a proxy
- disadvantage
	- proxies can monitor the user traffic 
	- evil twin attack
