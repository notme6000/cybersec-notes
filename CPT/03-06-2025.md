### gobuster

- used to enumerate directories
- developed using __golang__
- cli tool
- `gobuster dir -u http://<ip> -w <path to the wordlist>`
- common world list __/usr/share/wordlists/dirb/common.txt__
### enum4linux
- used to enum the smb username, fileshares
- `enum4linux -a <ip>`
### smbclient
- used for find the files in the smb server
- used like ftp
- `smbclient //<ip>/<file>`
### hydra
- used to crack password
- `hydra <option> <username/usernamelist> <option> <password/passwordlist> <ip>`
