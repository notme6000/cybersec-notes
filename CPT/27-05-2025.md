### google dorking
- using google search engine to find info about something with certain queries and keywords and search techniques
- `filetype:pdf` searches the web for pdf file
- `"content"` searches the keyword content in the web
- `inurl:<word>` search for the url with the word
- `site:<url>` search for the specific site
- `intext:<keyword>` search for specific keyword
- we can combine and use different methods to search
