### nmap
- #### xmas scan
	- `nmap -sX <ip>`
	- sends three flags
		- FIN - finish
		- URG - urgent
		- PSH - push
	- does not get response for the reques/packets
- #### udp scan
	- `nmap -sU <ip>`
	- to scan a UDP port
- #### icmp ping
	- `nmap -sn -PE <ip>`
	- to ping the server ip with icmp packets- #### tcp syn ping
	- `nmap -sn -PS <ip>`
	- uses __TCP SYN__ packets for pining the server.
	- used if the server blocks the __ICMP__ packets.
### nmap script engine
- The built-in script that can be used to enumerate the vulnerability of  a system.
	- #### ftp-anao
		- `nmap -p 21 --script ftp-anon <ip>`
		- checks for the vulnerability anonimous login in the port 21 ftp
	- #### smb (server message block)
		- `nmap -p 445 --script smd-enum-shares <ip>`
		- network message sharing protocol
		- linux - samba server
		- utility : file share, remote login, logging and user authentication

	- #### ssh auth methods
		- `nmap -p 22 --script ssh-auth-methods <ip>`
		- give the detailed method for authentication for the ssh
	- #### ssh brute force
		- `nmap -p 22 --script ssh-brute --script-args userdb=user.txt,passdb=pass.txt <ip>`
		- used bruteforce method to cract ssh auth.
	- #### anti firewall detection 
		- `nmap -p 80 --source-port 53 <ip>`
		- the firewall will see it as a legitimate for port 53
	- #### fragment packets scan
		- `nmap -f <ip>`
		- sends the packets in fragments to hide the nmap scan.
