 


## Level 1: Basic (Foundational)

Focus on learning your tools (**Nmap, Gobuster, Nikto**) and recognizing common "low-hanging fruit" like default credentials or simple SUID binaries.

1. **Kioptrix: Level 1 (#1)** – The classic "starter" box.
    
2. **Kioptrix: Level 1.1 (#2)** – Focuses on SQL Injection and basic web shells.
    
3. **DC: 1** – Great for learning CMS (Drupal) enumeration. ->  _completed_
    
4. **DC: 2** – Teaches you about password wordlists and non-standard SSH ports. ->  _completed_
    
5. **Basic Pentesting: 1** – Practice brute-forcing and SSH key exploitation.
    
6. **Mr. Robot: 1** – Based on the show; heavy focus on WordPress and directory discovery.
    
7. **Billu: b0x** – Excellent for practicing LFI (Local File Inclusion) and file uploads.
    
8. **Zico2** – Teaches you how to pivot from a web vulnerability to a user shell.
    
9. **Tr0ll: 1** – A "troll" box that teaches you to handle "rabbit holes" (fake leads).
    
10. **Lame** – Though originally a HackTheBox classic, it is available on VulnHub and focuses on the infamous Samba "SambaCry" vulnerability.
    



## Level 2: Intermediate (OSCP Style)

These machines are more realistic. You will often need to find a way to move "laterally" (from one user to another) before you can hit the root.

1. **Stapler: 1** – A massive machine with 12+ ways in. Best for practicing thorough enumeration.
    
2. **SickOs: 1.2** – Teaches you how to bypass firewall restrictions (egress filtering).
    
3. **VulnOS: 2** – A standard intermediate box involving a vulnerable CMS and kernel exploits.
    
4. **PwnLab: init** – Focuses on PHP filter tricks and database manipulation.
    
5. **Kioptrix: 2014 (#5)** – Introduces you to Free BSD and more complex web-based entry points.
    
6. **HackLAB: Vulnix** – No web server here! You must rely on older network services (NFS, SMTP).
    
7. **IMF: 1** – A "hidden" challenge series that mirrors CTF style but with realistic web flaws.
    
8. **SkyTower: 1** – Focuses on SQL Injection and proxying your traffic.
    
9. **Lord of the Root: 1.0.1** – Designed specifically for OSCP prep with a focus on Buffer Overflows.
    
10. **FristiLeaks: 1.3** – Involves decoding base64, finding hidden files, and clever privesc.
    



## Level 3: Advanced (Hardcore)

Expect custom binaries, restricted shells, and multi-step exploitation chains. These require deep patience and advanced scripting/debugging skills.

1. **Brainpan: 1** – The gold standard for practicing manual **Buffer Overflow** (BoF).
    
2. **Empire: Breakout** – Features custom shells and advanced privilege escalation.
    
3. **Scream (/dev/random)** – Hardcore Linux kernel exploitation and privilege escalation.
    
4. **Temple of Doom** – Requires high-level logic and exploitation of obscure Node.js vulnerabilities.
    
5. **Pinky's Palace: v1** – Intense web-focused box with multiple layers of security.
    
6. **Doubletrouble: 1** – Two machines in one; you must compromise the first to reach the second.
    
7. **Vikings: 1** – Focuses on lateral movement and decoding obscure file formats.
    
8. **Hacksudo: Thor** – A modern machine focusing on unique privilege escalation techniques.
    
9. **Kioptrix: Level 1.2 (#3)** – While numbered low, the manual exploitation required is significantly tougher than Level 1.
    
10. **pWnOS: 2.0** – An older but "fiendishly tricky" box that involves chaining multiple exploits.
