## Iot Concepts and Attacks
- Iot is a web of connected devices, made possible by the intersection between machine-to-machine communications and big data analytics.
- Attackers implement various techniques to launch attacks on target IoT devices or networks.
- IoT threats in relation to basic types of IoT attack vectors and techinques, including DDoS attacks, attacks on HVAC systems, rolling code attacks, BlueBorne attacks and jamming attacks.
## what is the IoT ?
- The internet of things, also known as the internet of everything, refers to computing devices that are web-enabled and have the capability of sensing, collecting, and sending data using sensors, and the communication hardware and processors that are embedded within the device.
## How IoT works ?
- _sensing technology_ : sensors embedded in the devices sense a wide variety of info from their surroundings, including temperature, gases, location.
- _IoT Gateway_ : Gateways are used to bridge the gap between an IoT device and the end-user.
- _Cloud server/Data Storage_ : Cloud is used to store the data, and sometimes process and then transmit to user.
- _Remote control using Mobile app_ : The end-user uses remote controls such as mobile phones installed with a mobile app to monitor and control data and take a specific action on IoT devices.
## IoT Architecture
- _Edge technology layer_ : This layer consists of hardware components, including sersors, RFID tags, readers and other soft seonsors and the device itself.
- _Access Gateway layer_ : This layer helps to bridge the gap between two endpoints, such as a device and a client.
- _Internet layer_ : This is a crucial layer as it serves as the main component in carrying out communication between two endpoints, such as device-to-device, device-to-cloud etc.
- _Middleware Layer_ : This layer operates in two-way mode. This layer sits in the middle of the application layer and the hardware layer, this behaving as an interface between these two layer. Function - data management, device management etc.
- _Application Layer_ : Responsible for the delivery of services to the relevant users from different sectors, including building, industrial, manufacturing etc.
## IoT Technologies and Protocols
- __Short-Range Wireless Communication__
	- _Bluetooth Low Energy (BLE)_ : bluetooth smart is a wireless personal area network.
	- _light-fidelity (Li-Fi)_ : Li-Fi is a visible light communications (VLC) system that uses light for data transfer at a very high speed of 224 Gbps.
	- _Near Field Communication (NFC)_ : NFC is a type of short-range communication that uses magnetic field induction to enable communication between two electronic devices, used in contactless mobile payment.
	- _QR Codes and Barcodes_ : These codes are machine-readable tags that contains info about the product or item to which they are attached.
	- _Radio-Frequency Identification (RFID)_ : RFID sotes data in tags that are read using electromagnetch fields.
	- _WIFI_ : Wi-Fi is a technology that is widely used in wireless local area network (LAN).
- __Medium-Range Wireless Communication__
	- _Halow_ : This is another variant of the Wi-Fi standard; it provides an extended range, making it usefull for communication in rural areas.
	- _LTE-Advanced_ : LTE-Advanced is a standard for mobile communcation that provides enhancement to LTE.
	- _6LoWPAN_ : IPv6 over low-power wireless personal area network (6LoWPAN) is an internet protocol used for communication between smaller and low-power devices with limited prcessing capacity, such as various IoT devices with limited processing capacity.
- __Long-Range Wireless Communication__
	- _LPWAN_ : Low power wide area network is a wireless telecommunication network, designed to provide long-range communication between two endpoints.
		1. LoRaWAN
		2. Sigfox
		3. Neul
	- _Very small Aperture Terminal_
	- _Cellular_
	- _MQTT_
- __Wired Communication__
	- _Ethernet_ : ethernet is the most commonly used type of network protocol today. It is a type of LAN.
	- _Multimedia over Coax Alliance ( MoCA)_
	- _Power-Line Communication (PLC)_
